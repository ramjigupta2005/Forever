# 🚀 Forever - Full Stack E-commerce Website

## 📌 Project Overview

Forever is a **full-stack e-commerce web application** built using the MERN stack.
It includes a customer-facing frontend, an admin dashboard, and a backend API.

---

* 🧑‍💻 Admin Panel:
  https://forever-admin-six-theta.vercel.app/

* 🛒 Frontend Website:
  https://forever-frontend-sandy-tau.vercel.app/

* ⚙️ Backend API:
  https://forever-backend-rho-cyan.vercel.app/

## 🏗️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Tools

* JWT Authentication
* Cloudinary

---

## ✨ Features

### 👤 User Features

* User authentication (Login / Signup)
* Browse products
* Add to cart
* Place orders

### 🛠️ Admin Features

* Add / Update / Delete products
* Manage orders
* Dashboard access

### ⚙️ Backend Features

* REST APIs
* Secure authentication (JWT)
* Database integration

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/ramjigupta2005/Forever.git
cd Forever

---

### 2️⃣ Setup Backend

cd backend
npm install
npm run dev

---

### 3️⃣ Setup Frontend

cd frontend
npm install
npm run dev

---

### 4️⃣ Setup Admin Panel

cd admin
npm install
npm run dev

---

## 🔑 Environment Variables

Create a `.env` file in backend:

MONGODB_URI=your_mongodb_url
JWT_SECRET=your_secret_key
CLOUDINARY_API_KEY=your_key

Frontend `.env`:

VITE_BACKEND_URL=https://forever-backend-rho-cyan.vercel.app
